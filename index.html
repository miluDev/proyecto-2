Transiciones

Una transición en CSS nos permite otorgar un tiempo entre el estado inicial y final de un elemento, logrando que estos cambios no ocurran de manera instantánea, lhaciendo de estas transiciones, una animación.

Debemos tene en cuenta que los elementos inline no se pueden animar, que todos los elementos tienen un estado inicial, que no todas las propiedades son animables, que debe haber un detonante para que una tansición suceda, es decir, no se ejecuta por sí sola.

Animaciones

Las animaciones permiten animar la transición de un estilo CSS a otro estilo, a diferencia de as transiciones no necesitan un detonador para iniciar y pueden ser ejecutadas cierto número de veces o de manera infinita.

Las animaciones requieren recursos del sistema, por ello es importante conocer la renderización de los elementos, por ejemplo, transform y opacity ocupan pocos recursos.
